# Pokemon - dokumentáció készítésének automatizálása

## Leírás:

Ez a projekt egy Pokémon témájú webalkalmazás, amely célja, hogy különböző Pokémon karaktereket jelenítsen meg és azok adatait kezelje. A felhasználók böngészhetik a Pokémonokat, megtekinthetik a tulajdonságaikat, illetve interakcióba léphetnek velük az alkalmazás funkcióinak segítségével.
A projekt oktatási céllal készült, és bemutatja a frontend fejlesztés alapjait, valamint API használatát.

## Használt technológiák:

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API / async-await

## Funkciók:

- Pokémonok listázása
- Pokémon adatok megjelenítése (név, típus, kép stb.)
- Keresés vagy szűrés (ha van ilyen funkció)
- API-ból történő adatlekérés
- Reszponzív design

## Dokumentáció

A projekt JSDoc segítségével dokumentálva lett.

A dokumentáció generálásához használd:

bash
npm run doc
  

## Fejlesztési lehetőségek:

- Több szűrési opció hozzáadása
- Pokémon részletes nézet bővítése
- LocalStorage használata kedvencekhez
- Animációk és UX javítás
